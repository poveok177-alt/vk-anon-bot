# 🤫 VK Анонимный бот — Инструкция по установке

## Что нужно (всё бесплатно и работает из РФ)

| Что        | Где взять          | Зачем                  |
|------------|--------------------|------------------------|
| VK-группа  | vk.com/groups      | Бот живёт в группе     |
| Bothost.ru | bothost.ru         | Хостинг (бесплатно)    |
| SQLite     | встроен в Python   | База данных            |

---

## ШАГ 1 — Создай группу ВКонтакте

1. Зайди на **vk.com** → «Сообщества» → «Создать сообщество»
2. Выбери «Публичная страница» или «Группа»
3. Перейди в **Управление → Работа с API**
4. Нажми **«Ключи доступа»** → **«Создать ключ»**
5. Отметь все галочки (особенно «Сообщения»), сохрани
6. **Скопируй токен** — он начинается с `vk1.a.`

Затем обязательно:
- **Управление → Сообщения** → включи «Сообщения сообщества»
- **Управление → Боты** → включи «Возможности ботов»
- **Управление → Боты → Long Poll API** → включи, версия API = **5.199**

### Где найти ID группы:
Открой страницу группы → в адресной строке будет `vk.com/club123456789` — число и есть ID.

### Где найти свой VK ID:
Открой **vk.com/usrid** — в строке URL увидишь число.

---

## ШАГ 2 — Зарегистрируйся на Bothost.ru

1. Зайди на **bothost.ru**
2. Нажми «Регистрация» (можно через VK)
3. После входа нажми **«Создать бота»**
4. Выбери **Python** как язык
5. Выбери **бесплатный тариф**
6. Назови бота как угодно → «Создать»

---

## ШАГ 3 — Загрузи файлы бота

1. В панели Bothost открой своего бота → **«Файлы»**
2. Загрузи все файлы из архива:
   - `main.py`, `config.py`, `database.py`
   - `keyboards.py`, `states.py`, `anon_message.py`
   - `admin.py`, `tasks.py`, `requirements.txt`

---

## ШАГ 4 — Создай файл .env

В файловом менеджере Bothost создай файл `.env`:

```
VK_TOKEN=vk1.a.ВАШ_ТОКЕН_ГРУППЫ
VK_GROUP_ID=123456789
ADMIN_VK_ID=123456789
BOT_LINK=https://vk.me/ваша_группа
DB_PATH=./data/bot.db
```

Замени все значения на свои (см. Шаг 1).

---

## ШАГ 5 — Настрой запуск

1. В настройках бота найди **«Команда запуска»**
2. Вставь: `python main.py`
3. Нажми **«Запустить»** (▶️)

В логах должно появиться: `✅ VK-бот запущен (Long Poll)`

Зависимости из `requirements.txt` Bothost установит **автоматически**.

## Команды администратора (пиши боту в ЛС)

| Команда                  | Что делает                    |
|--------------------------|-------------------------------|
| `/admin`                 | Панель администратора         |
| `/stats`                 | Статистика бота               |
| `/ban 123`               | Забанить пользователя         |
| `/unban 123`             | Разбанить                     |
| `/userinfo 123`          | Инфо о пользователе           |
| `/broadcast Текст`       | Рассылка всем                 |
| `/fakemsg 123 Текст`     | Анонимка одному               |
| `/ad`                    | Панель рекламы                |
| `/ad_on` / `/ad_off`     | Вкл / выкл рекламу            |
| `/ad_text Текст`         | Текст рекламы                 |
| `/ad_url https://...`    | Ссылка рекламной кнопки       |
| `/ad_btn Текст`          | Текст кнопки                  |

## Частые проблемы

**Бот не отвечает:**
- Проверь что в группе включены Сообщения и Long Poll API
- Убедись что токен правильный в `.env`
- Смотри логи в панели Bothost

**«Can't send messages to this user»:**
Пользователь должен сам написать первым в группу — VK требует разрешения.

**База данных не создаётся:**
Измени в `.env` на `DB_PATH=./bot.db` (без подпапки).

```
├── .env              — токены и настройки (не публикуй!)
└── data/bot.db       — база данных (создаётся сама)
```
